== Description ==

Displays the Alexa traffic rank in the sidebar of your blog via widget interface or anywhere else via function call. The Plugin comes with 5 different layouts.

Browse more [Wordpress Plugins](http://www.fliptel.de/wordpress-plugins "Wordpress Plugins") by [Fliptel](http://www.fliptel.de "Fliptel").

== Installation ==

1. Unpack the zipfile alexarank-X.y.zip
1. Upload folder `alexarank` to the `/wp-content/plugins/` directory
1. Make cache directory writeable chmod 777
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php if(function_exists('alexarank_display'))alexarank_display(); ?>` in your template or use the sidebar widgets.
